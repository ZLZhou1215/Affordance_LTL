From Ambiguous Language to Verifiable Plans: Integrating Formal Synthesis and Dynamic Affordance Reasoning

Paper Code Video Dataset Hugging Face Demo

Abstract

Enabling robots to execute ambiguous natural language commands in semi-structured environments remains a fundamental challenge in robotics, requiring the translation of high-level human intent into verifiable physical actions. Existing approaches fail to bridge this semantic-physical gap: symbolic methods demand exhaustive manual state specification and cannot adapt to environmental changes; learning-based methods lack formal guarantees and generate physically infeasible plans.

We present a hierarchical reactive planning framework that integrates formal verification with dynamic perception through three synergistic components. First, VLPGen dynamically synthesizes minimal, task-relevant Linear Temporal Logic over Finite Traces (LTLf) specifications from visual-linguistic context, circumventing the state-space explosion inherent in traditional formal methods. Second, ContextualAfford enables zero-shot functional equivalence reasoning by identifying substitute objects based on physical affordances rather than semantic categories‚Äîrecognizing, for instance, that a ruler's edge can cut soft dough when a knife is unavailable. Third, a hierarchical execution architecture maintains semantic consistency through tiered recovery mechanisms tailored to failure modes: functional substitution for object unavailability, local reactive synthesis for path deviations, and global replanning for specification violations.

Extensive experiments on a Franka Panda manipulator executing long-horizon tasks in dynamic environments demonstrate our framework's effectiveness. We also introduce ContextaFF, a benchmark dataset of 550 images with pixel-level affordance annotations and natural language instructions for advancing language-grounded robotics research.

Framework Architecture

Framework Overview: An overview of our hierarchical reactive planning framework. The VLPGen module grounds user intent into a formal LTLf specification, which guides the Hierarchical Reactive Planner. The planner leverages real-time perceptual information from the ContextualAfford module to ensure robust and adaptive execution.

ContextaFF Dataset

ContextaFF is a comprehensive benchmark dataset for context-aware affordance understanding in robotic manipulation. The dataset comprises 550 real-world scene images, each paired with natural language instructions and corresponding pixel-level affordance annotations, enabling precise evaluation of context-sensitive affordance grounding methods.
